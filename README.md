# SnowTricks

SnowTricks est une plateforme dédiée aux passionnés de snowboard qui souhaitent partager, découvrir et apprendre de
nouvelles figures.
Un spot communautaire où chacun peut enrichir la base de tricks… ou simplement s’inspirer avant la prochaine session.

## Fonctionnalités

- Partage de figures : ajoutez vos tricks avec description, images et vidéos.
- Recherche avancée : par nom, type de figure ou niveau de difficulté.
- Section commentaires : échangez autour des tricks, donnez vos conseils, vos variantes, vos fails (on juge pas).
- Inscription & Connexion : espace membre sécurisé.
- Modération : gestion des contenus pour garder un environnement propre et sain.

## Installation

Clonez le projet :

```bash
git clone git@github.com:leomoille/snowtricks.git
```

Installez les dépendances PHP :

```bash
composer install
```

Lancement du serveur :

> Assurez-vous d’avoir installé symfony-cli, puis démarrez le serveur local :

```bash
symfony serve
```

Vous pouvez ensuite accéder au projet via l’URL indiquée dans le terminal.
